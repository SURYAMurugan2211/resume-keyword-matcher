## üé® Features Breakdown

### Weighted Scoring System
- **Skills** (40%): Programming languages, soft skills
- **Technologies** (30%): Frameworks, platforms, databases
- **Tools** (20%): Development tools, software
- **Job Roles** (10%): Titles, role-specific terms

### PDF Processing
- Client-side text extraction
- Multi-page support
- Text normalization
- Handles hyphenated words
- Error handling for corrupted/encrypted PDFs

### Smart Matching
- Case-insensitive fuzzy matching
- Handles variations (JavaScript vs javascript)
- Substring matching
- Bidirectional keyword detection
